# Prime Number
## Description
This project is a Python program that demonstrates the use of while loop in Python. A while loop repeatedly executes a block of code as long as **a specified condition is True**. It is useful when the number of iterations is not fixed or depends on a condition.

A **Prime Number** is a natural number that has exactly two factors: **1 and itself**.
## Syntax
```python
while condition:
   # Code to execute if condition is True
```

## Working of the Code
1. The program asks the user to input a number in the variable 'num'.
2. The variable 'i' is inialized with '2'.
3. The variable 'prime' is initialized with 'True. 
4. The 'while' loop runs as long as 'i' is less than 'num'.
5. If the number is exactly divisible by any value of i, prime is set to False.
6. The break statement immediately stops the loop because the number is already known to be not prime.
7. If no divisor is found, prime remains True.
8. After the loop ends, the program checks:
- If prime is True and the number is greater than 1, it prints "Prime Number".
- Otherwise, it prints "Not a Prime Number".
